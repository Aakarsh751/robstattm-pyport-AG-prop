# Example R scripts — **RobStatTM** (not PCRA)

These `.R` files accompany the book *Robust Statistics: Theory and Methods* and the **[RobStatTM](https://cran.r-project.org/package=RobStatTM)** R package. They **`library(RobStatTM)`** and reproduce figures/tables from that book (e.g. flour, shock, oats).

**They do not belong to PCRA.**  
The portfolio / PCRA Chapter 2 demo lives under **`pcra/r/`** (e.g. `Ch_2_Foundations_Demo.R`, `run_ch2_demo.R`).

| This folder | PCRA folder (`pcra/`) |
|-------------|------------------------|
| Robust stats, regression, book datasets | Efficient frontiers, GMV, CRSP data, `library(PCRA)` |

Run from project root with `setwd()` to `ProfDM_Rproject`, then e.g. `source("robstattm/examples-scripts/shock.R")`.

See also: `robstattm/docs/README_Scripts_Checklist.md` (checklist) and `Examples Scripts and Data Sets Guide` PDF in `robstattm/docs/`.
